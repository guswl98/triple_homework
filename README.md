# 신현지 트리플 프론트엔드 과제

## 프로젝트 구성

```
├─ pulic
│ ├─ index.html     : DOM root
│ └─ manifest.json
├─ src
│ ├─ assets         : imgs
│ ├─ components     : React components
│ │  └─ common      : Common components
│ ├─ styles         : css files
│ ├─ utils          : Functions
│ └─ index.tsx
├─ tsconfig.json
├─ package.json
├─ .eslintrc.js
└─ .prettierrc
```

## Dependencies

- Git
- yarn or NPM

## Clone the Repository

```
$ git clone https://github.com/guswl98/triple_homework.git
```

## Execute

```
$ yarn start
```

or

```
$ npm start
```
